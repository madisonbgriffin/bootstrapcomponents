Windows 7


Windows 7 made major advancements in the software world. The software was launched on July 22, 2009. The biggest aspect of Windows 7 is that it ran faster and more reliably than the versions before it. However, a setback of this is that Windows 7 required a 64-bit hardware to run smoothly and efficiently, forcing consumers to purchase new hardware. The user interface was changed and improved in this new version. It was an interface free of clutter and pointless applications. One downfall of Windows 7 is that it was slightly inconvenient considering users are recommended to get a new hardware for the version. Although using the same hardware and updating it is possible, it is inefficient and time-consuming. Windows 7 also requires 5GB of extra space in order to upgrade directly from Windows XP. Version 7 has the same User Access Controls as Windows Vista, but it is smoother, and it allows for UAC messaging to be disabled. Windows 7 also supported virtual hard drives. A front-end feature that was improved was the taskbar. The taskbar gave users the option to pin their most frequently used programs or apps. Windows 7 also got rid of many features of previous versions, such as Windows Mail, Windows Calender, Windows Movie Maker, the start menu, and even more. Windows 7 introduced the classic file library, where users could bundle files from all locations on the hard drive into folders or designate the files to quick access for convenience.
