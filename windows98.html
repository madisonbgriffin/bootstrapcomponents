Windows 98 Second Edition


Following the release of the Windows 95 product and Windows 98 First Edition, Windows 98 Second Edition focused greatly on web integration. Arriving less than a year after the first edition, the software was changed to accommodate for the use of Internet Explorer. The new version of Internet Explorer provided a new user-friendly experience, so it was recommended for consumers to purchase Second Edition rather the first to get the full extent of Internet Explorer 5. It also included support for USB. Windows 98 First Edition resulted in a blue screen when a USB device was plugged in, so Windows 98 Second Edition resolved this problem on May 5, 1999. In addition to USB support, Windows 98 SE provided sound/audio card support through the Windows Driver Model. This product was the last OS to include to MS-DOS 7.10. Compression Agent and DriveSpace were removed from the System Tools Folder. The product also included an updated version of Windows Media Player, so users could now play Windows Media Files, but not MP3s or QuickTime and RealMedia Files. The software was sold as an installation or an upgrade to previous Windows versions. It was sold at a price of $109.00, making it rather affordable for the public. Overall, Windows 98 Second Edition improved and enhanced the features of Windows 98 First Edition. It provided new tools for a more user-friendly experience. However, the software was still widely criticized as its predecessors, forcing its developers to continue creating new features and enhancing current ones.
