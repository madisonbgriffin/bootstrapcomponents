Windows 10


Windows 10 is the latest version of Microsoft Windows. One distinction between Windows 10 and other versions is that Windows 10 can run on a tablet, in addition to desktops and laptops. Windows 10 can tell the difference easily between tablet or desktop based on whether or not a mouse and keyboard is being used. Windows 10 reintroduced the start menu, despite it previously being taken away in Windows 7. The start menu appears in the bottom left of the screen. It was modified for a more aesthetic look and traditional role. It was expanded to create a larger view of possible apps. Also, Windows 10 introduced Microsoft Edge. For decades, Internet Explorer was the primary browser, but Microsoft Edge had more to offer. Windows 10 also includes a virtual assistant, called Cortana, that is activated by voice command. Cortana can complete simple tasks such as searching the web for a question or pulling certain files. The new software allows user to sort away unneeded windows in virtual desktops that can be pulled back up later. This promotes multi-tasking and organizations as users sort windows into different screens. Users can easily shift back and forth between virtual desktops with the new Task View button. The new Action Center in Windows 10 was updated to allow users to access more settings in a convenient spot. It also displays important notifications. Windows 10 provides a more efficient computer experience for all who use it. It builds upon the fails of its predecessors and sets a tone for future software technology.
